### Development
#### Getting Started  

Clone repo using

```
git clone https://github.com/roeintheglasses/apollo.git
```

#### Install yarn dependencies

cd within the repo directory

```
yarn install
```
#### Scripts 

##### Start

within the repo directory use

```
yarn start
```
to start the expo server and run the expo development server


##### Build

within the repo directory use

```
expo build
```
or use

```
expo build:android:
```
for android and
```
expo build:ios
```
for the ios build.
Make sure to modify the app.json file as per your needs.

##### The web app will run at port 3000 by default at localhost.

```
http://localhost:3000/

```
