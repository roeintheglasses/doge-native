### Development
#### Getting Started  

Clone repo using

```
git clone https://github.com/roeintheglasses/doge-native.git
```

#### Install yarn dependencies

cd within the repo directory

```
yarn install
```
#### Scripts 

##### Start

within the repo directory use

```
yarn start
```
to start the expo server and run the expo development server


##### Build

within the repo directory use

```
expo build
```
or use

```
expo build:android:
```
for android and
```
expo build:ios
```
for the ios build.
Make sure to modify the app.json file as per your needs.
